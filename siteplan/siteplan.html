Site Name

MovieVerse

The name MovieVerse represents a space (or universe) where users can explore and discover movies from different genres and eras. It is short, memorable, and fits the entertainment theme of the website.

Optional domain: movieversehub.com (not required for this project)

Site Purpose

The purpose of MovieVerse is to provide an engaging platform for users to explore and learn more about movies. The website will feature a collection of movie titles displayed dynamically using data fetched from a public API. Users can filter or sort movies by genre, view detailed information in modals, and save their favorites using Local Storage. The site will demonstrate responsive design, accessibility, and interactivity using HTML, CSS, and JavaScript.

Scenarios

  • Which movies are trending right now or most highly rated?
  • Can I view detailed information (like release date, genre, and description) about a specific movie?
  • How can I save my favorite movies to view later?
